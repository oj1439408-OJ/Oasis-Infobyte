# Data Analytics Internship Projects

This repository contains the work I completed during my Data Analytics Internship. The main goal of this internship was to gain practical experience in working with real datasets, performing data analysis, and building machine learning models.

During the internship, I used Python along with libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn to clean data, explore datasets, visualize patterns, and build predictive models.

One of the main projects in this repository is the Wine Quality Prediction project. In this project, I analyzed a dataset containing different chemical properties of wine and built a Random Forest model to predict wine quality. I also performed exploratory data analysis using visualizations like pair plots and density plots, and handled class imbalance using SMOTE.

Another project focuses on Credit Card Fraud Detection. In this project, I built a Logistic Regression model to detect fraudulent transactions. The model was evaluated using metrics such as accuracy, confusion matrix, classification report, and ROC-AUC score.

Through these projects, I gained hands-on experience in data preprocessing, exploratory data analysis, machine learning model building, and performance evaluation.
